Site Plan – Personal Portfolio

Site Name

Carlos Valdés Portfolio

This site name directly represents the owner and subject of the website. It is professional, clear, and appropriate for a personal brand.

Site Purpose

The purpose of this website is to present my professional profile, technical skills, and experience in logistics, international trade, and engineering. It will also showcase selected projects and include a small dynamic API feature.

Scenarios

  • What experience does Carlos have in international trade and ocean freight logistics?
  • How can I view Carlos' skills and projects?
  • How can I contact Carlos for professional opportunities?

Color Scheme

  • Primary Color (#1A4D8F): Headers, navigation, buttons
  • Secondary Color (#F2F5F9): Background sections, content containers
  • Accent Color (#E67E22): Highlights, links, CTAs

Typography

  • Poppins: Headings and navigation
  • Roboto: Body text and paragraphs
  • Roboto Mono: Optional for code snippets or tags
